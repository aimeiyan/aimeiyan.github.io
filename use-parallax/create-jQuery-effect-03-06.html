Bubble Popup 效果

jQuery Bubble Popup是一个漂亮的弹出气泡或Tooltips，高度可配置，支持为任意DOM元素创建Bubble Popup，可以在目标元素的左右，上下位置弹出气泡。支持在运行时更改外观，异步加载数据，自定义弹出事件等。此动画没有借助插件，全部用jQuery代码编写。
