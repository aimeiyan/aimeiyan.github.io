Font Awesome 4.1

Font Awesome是矢量图标，可以通过CSS样式改变其大小、颜色、阴影等，另：自 Font Awesome 4.1 版本，所有的图标都需要两个class类：基础类fa和代表图标专门的类。例如： fa fa-adjust
进一步了解可以访问该网址 http://fortawesome.github.io/Font-Awesome/icons/
需要引入<link href="css/font-awesome.min.css" rel="stylesheet">文件

叠加多个图标，父亲节点需增加fa-stack， 里层图标class类 fa-stack-1x, 外层图标class类 fa-stack-2x。 增加类 fa-inverse可以得到反色效果，图标的大小可以调节。

类fa-border可以给图标添加边框效果。
